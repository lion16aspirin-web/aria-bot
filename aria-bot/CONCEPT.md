# ARIA — AI Discord Bot для Star Citizen організації

## Що це таке
Discord бот з AI особистістю для управління SC організацією.
ARIA = Automated Response & Intelligence Assistant
Стиль комунікації — бортовий AI корпорації, Star Citizen всесвіт.

---

## Модулі

### 1. Статуси гравців
Кожен оновлює свій статус командою:
```
!статус торгівля Microtech→ArcCorp
!статус mining Yela
!статус pvp Aaron Halo
!статус офлайн
```
Канал #org-статус автоматично оновлюється і показує хто онлайн, де, що робить.
Статус скидається автоматично коли гравець офлайн в Discord.

### 2. Інфо по грі
Підключення до SC Trade API (UEX Corp або SC Trade Tools) для актуальних цін.
```
@ARIA ціна золото Microtech
@ARIA порівняй Cutlass Black і Avenger Titan
@ARIA найвигідніший маршрут бюджет 50k Cutlass
```

### 3. Місії та івенти
Створення місії:
```
!місія створити "Рейд на Jumptown"
```
Бот розгортає картку з ролями, слотами, часом старту.
Гравці реєструються на ролі.
Нагадування автоматично за 1 годину до старту.
Після місії — логування результату.
Коли місія стартує — статуси учасників оновлюються автоматично.

### 4. AI особистість (ARIA)
Claude API як мозок бота.
Знає контекст організації: статуси, склади, місії, історію.
Розмовний інтерфейс — не тільки команди а і вільний чат.
Пам'ятає хто які ролі грає, хто з ким в команді, історію місій.
```
@ARIA підготуй список для місії на Jumptown, потрібні 2 пілоти і 3 стрільці
```
ARIA аналізує хто онлайн, хто вільний, хто близько по локації — і пропонує склад.

### 5. Менеджер складу
Реєстрація майна вручну:
```
@ARIA додай до мого складу: 50 медпаків, 2 Rifle AR-55
локація: Lorville

@ARIA додай до корп складу: паливо 2000 units
локація: Port Tressler
```
При плануванні рейду ARIA знає:
- хто що має
- де це знаходиться
- хто зараз онлайн і може привезти
- скільки часу займе доставка

Видає логістичний план: хто що бере і везе до точки збору.

## Технічний стек

```
Discord bot — discord.py
AI — Claude API (claude-sonnet-4-20250514)
База даних — SQLite (статуси, склади, місії, гравці)
SC дані — UEX Corp API або SC Trade Tools API
Деплой — Railway або Render (безкоштовний tier)
Мова — Python 3.11+
```

## Структура проекту

```
aria-bot/
├── main.py              # точка входу, ініціалізація бота
├── config.py            # токени, API ключі, налаштування
├── database.py          # SQLite, всі моделі даних
├── modules/
│   ├── status.py        # модуль статусів гравців
│   ├── missions.py      # модуль місій і івентів
│   ├── inventory.py     # модуль складів
│   ├── intel.py         # модуль інфо по грі (SC API)
│   └── aria.py          # Claude API, AI особистість
├── utils/
│   └── formatters.py    # форматування повідомлень Discord
└── requirements.txt
```

## База даних — схема

### players
- discord_id, username, status, location, activity, last_seen

### inventory
- id, owner_id, owner_type (personal/corp), item_name, quantity, location, updated_at

### missions
- id, name, creator_id, location, start_time, status, roles_json, participants_json, log

### mission_roles
- mission_id, role_name, slots_total, slots_filled, participants
